## Gemini Extension Sample

To use this repo, you need to create a new repo with it that uses your deployment server.

Once done, replace the `http://localhost:8000/mcp` in the `gemini-extension.json` file with the deployment URL.

Then, run:

```bash
gemini extension install https://github.com/<your_username>/fastmcp-gemini-ext
```

To install the extension to Gemini CLI.
